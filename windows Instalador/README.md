# Instalador de Astro-Nex para Windows

Esta carpeta contiene el instalador de Astro-Nex para Windows 8 y Windows 10
de 64 bits.

## Instalación recomendada

1. Ejecuta `Astro-Nex-Setup.exe`.
2. Sigue los pasos del asistente de instalación.
3. Abre Astro-Nex desde el acceso que el instalador crea en Windows.

El instalador incluye Python, GTK3, PyGObject, Pycairo, la extensión
astronómica `_pysw` y los recursos necesarios. No es necesario instalar esos
componentes por separado.

## Variante portable

La subcarpeta `Astro-Nex` contiene una versión portable. Puede ejecutarse con
`Astro-Nex/Astro-Nex.exe`, pero debe conservarse completa: no muevas el EXE sin
la carpeta `_internal`.

Windows puede mostrar una advertencia al ejecutar un programa no firmado.
Usa únicamente una copia descargada de este repositorio oficial.
